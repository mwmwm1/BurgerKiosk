# (C# 코딩) 버거 키오스크
## 개요
-C# 프로그래밍학습
-1줄소개: 햄버거를 주문받는 키오스크 프로그램
-사용한플랫폼: -C#, .NET Windows Forms, Visual Studio, GitHub
-사용한컨트롤:
	-Label, ListBox, Button, pictureBox, CheckBox, RadioButton
-사용한기술과구현한기능:
	-Visual Studio를이용하여UI 디자인
